Data Flow Diagrams (DFD)

Note: ArgoUML is primarily designed for UML diagrams (Use Case, Class, Sequence, etc.). DFDs are structured analysis tools best viewed as standard flowcharts. You can interact with the diagrams below.

Context Diagram (Level 0 DFD)

Shows the entire system as a single process interacting with external entities.
